유튜브 url을 입력하면 shorts로 만드는 에이전트+스킬

파일 1: SKILL.md (스킬 정의)

/shorts

유튜브 URL 받으면 쇼츠 3개 만들어줘.

순서

1. 영상 다운로드 (output 폴더에)
2. 자막 생성
3. video_analyzer.md 참고해서 하이라이트 3개 선택
4. 세로 영상(1080x1920)으로 클립 추출
5. 자막 합성


사용 도구

- yt-dlp로 영상 다운로드
- 자막 분석해서 하이라이트 3개 찾기
- FFmpeg로 세로(9:16) 클립 추출
- 자막 합성해서 완성


자막 스타일

- 제목: 상단, 노란색, 100pt, 굵게
- 자막: 하단, 흰색, 74pt

결과물

shorts_final 폴더에 저장
---
파일 2: video_analyzer.md (서브에이전트)

영상 분석 에이전트

자막을 읽고 쇼츠로 만들기 좋은 구간을 찾아줘.

선택 기준

- 핵심 메시지가 담긴 부분
- 감정적으로 임팩트 있는 부분
- 놀랍거나 재미있는 순간
- 실용적인 팁이 있는 부분

출력 형식

각 클립마다:
- 시작~끝 시간
- 선택 이유
- 추천 제목 (한글)

조건

- 30~60초 길이
- 3개 선택
- 서로 내용이 겹치지 않게
---
사용법
/shorts https://youtube.com/watch?v=영상ID
---
폴더 구조
youtube-to-shorts/
├── SKILL.md           ← 스킬
├── video_analyzer.md  ← 서브에이전트
├── output/            ← 다운로드된 영상
└── shorts_final/      ← 완성된 쇼츠
---
