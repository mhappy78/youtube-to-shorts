# YouTube to Shorts Converter

유튜브 영상을 자동으로 쇼츠(Shorts) 영상으로 변환하는 도구입니다.

## 기능

- 유튜브 URL 입력만으로 자동으로 쇼츠 영상 3개 생성
- AI 기반 하이라이트 구간 자동 선택
- 세로 영상(9:16 비율) 자동 변환
- 자막 자동 합성

## 사용법

```
/shorts https://youtube.com/watch?v=영상ID
```

## 작동 방식

1. **영상 다운로드**: yt-dlp로 영상 다운로드
2. **자막 생성**: 자막 파일 다운로드 또는 생성
3. **하이라이트 분석**: AI가 쇼츠로 만들기 좋은 구간 3개 선택
4. **세로 영상 추출**: FFmpeg로 9:16 비율 클립 추출
5. **자막 합성**: 클립에 자막 오버레이

## 필요 도구

- `yt-dlp`: 유튜브 영상 다운로드
- `ffmpeg`: 영상 편집 및 자막 합성

## 폴더 구조

```
youtube-to-shorts/
├── SKILL.md           # 스킬 정의
├── video_analyzer.md  # 하이라이트 분석 에이전트
├── guide.md           # 가이드 문서
├── output/            # 다운로드된 영상 (git 제외)
└── shorts_final/      # 완성된 쇼츠 (git 제외)
```

## 자막 스타일

- **제목**: 상단, 노란색, 100pt, 굵게
- **본문**: 하단, 흰색, 74pt

## 주의사항

- 각 쇼츠는 30~60초 길이로 제한됩니다
- 세로 화면에서 중요한 부분이 잘리지 않도록 자동 조정됩니다
- 자막은 가독성을 위해 최적화됩니다
